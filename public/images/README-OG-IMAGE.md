# Image Open Graph

L'image `photo-christine.jpg` (1200×630 pixels) est requise pour les aperçus sociaux (Open Graph, Twitter Cards).

## Création de l'image

1. Prendre une photo professionnelle de Christine Solignac
2. Redimensionner à 1200×630 pixels
3. Optimiser pour le web (JPEG, qualité 85-90%)
4. Sauvegarder comme `photo-christine.jpg` dans ce dossier

## Outils recommandés

- Photoshop / GIMP
- ImageMagick: `convert input.jpg -resize 1200x630^ -gravity center -extent 1200x630 -quality 90 photo-christine.jpg`
- En ligne: Canva, Photopea

## Format requis

- Dimensions: 1200×630 pixels (ratio 1.91:1)
- Format: JPEG
- Taille: < 1MB recommandé
- Contenu: Photo professionnelle de Christine Solignac avec fond neutre
